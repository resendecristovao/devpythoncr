# devpythoncr
Projeto para processo seletivo da Hashtag Treinamentos

Este é uma API feita em Python utilizando Flask e o PostGRE para o processo seletivo de desenvolvedor python da Hashtag Treinamentos.

O projeto possui 3 partes:
- Página de login e cadastro de usuários;
- Endpoint do webhook;
- Página onde é exibido o histórico de registros recebidos do webhook e as ações que cada um dos registros gerou.
